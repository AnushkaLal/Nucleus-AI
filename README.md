# Nucleus AI

Nucleus AI is an offline voice assistant prototype designed for educational support and everyday conversations, with a focus on accessibility in low-connectivity environments.

## Features
- Offline speech-to-text
- Local LLM inference
- Offline text-to-speech
- Short educational explanations
- General knowledge assistance
- Speech generation and conversational responses

## Tech Stack
- Python
- llama.cpp
- Whisper
- Piper TTS
- Electron
- Raspberry Pi 5 (target deployment)

## Current Status
- Voice interaction pipeline implemented
- Local inference working
- Raspberry Pi deployment in progress

Check out the [Design Specifications](design.md) and [Project Architecture](architecture.md).
